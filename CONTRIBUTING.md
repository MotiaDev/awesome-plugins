# Contributing to Awesome Motia Plugins

Thank you for your interest in contributing to the Awesome Motia Plugins list! We welcome contributions from the community.

## How to Add a Plugin

1.  **Search**: Ensure your plugin isn't already listed.
2.  **Fork**: Fork this repository.
3.  **Add**: Add your plugin to the `README.md` file in the appropriate category.
    *   Format: `- [Plugin Name](Link) - Description.`
    *   Keep descriptions concise and clear.
4.  **Pull Request**: Submit a Pull Request with a clear title (e.g., "Add plugin-xyz").

## Guidelines for Plugins

*   Plugins should be useful and relevant to the Motia ecosystem.
*   Links should be valid and accessible.
*   If your plugin is open source, please provide a link to the repository.

## creating a New Plugin

If you want to create a new plugin for Motia, please refer to the official resources:

*   [Motia Plugin Development Guide](https://github.com/MotiaDev/motia/blob/main/packages/docs/content/docs/development-guide/plugins.mdx)
*   [Plugin Example](https://github.com/MotiaDev/motia/tree/main/plugins/plugin-example) - A minimal example plugin to use as a template.

We encourage you to publish your plugins to npm so they can be easily installed by others, though it is not strictly required if you provide clear installation instructions.
